# XTMC量化交易系统

**自我进化型AI量化交易系统**

XTMC (eXponential Trading Mastery Core) 是一个专为网心云OES Plus设计的自我进化型AI量化交易系统。系统能够自动反思、规划、生成、测试并保存交易工具，实现持续自我提升。

## 核心特性

### 🧬 自我进化机制

XTMC采用六步自我进化循环:

1. **反思** - AI分析当前交易表现和能力缺口
2. **规划** - 识别需要的新工具/策略
3. **生成** - 自动编写代码实现新工具
4. **测试** - 验证新工具的有效性
5. **保存** - 将有效工具添加到工具库
6. **循环** - 持续执行进化周期(每6小时)

### 📊 交易功能

- **多交易所支持**: Binance、OKX等主流交易所
- **实时K线图表**: 专业级图表显示
- **AI交易信号**: 多因子综合分析
- **自动/手动交易**: 灵活的交易模式
- **风险管理**: 可配置的止损和仓位管理

### 🤖 AI功能

- **智能分析**: RSI、MACD、EMA、布林带、ADX、KDJ等
- **多工具协同**: 工具库中所有工具共同决策
- **自然语言对话**: Web界面AI聊天助手
- **持续学习**: 从交易历史中学习和改进

## 硬件要求

- **设备**: 网心云OES Plus (或其他Armbian设备)
- **CPU**: Amlogic A311D 或同等性能ARM处理器
- **内存**: 4GB RAM
- **存储**: 6GB+ (强烈推荐外接SATA硬盘)
- **系统**: Armbian (Debian/Ubuntu基础)

## 快速安装

```bash
# 1. 上传部署包到设备
scp xtmc-trading.tar.gz root@<设备IP>:/tmp/

# 2. 解压并安装
ssh root@<设备IP>
cd /tmp
tar -xzf xtmc-trading.tar.gz
cd xtmc-trading
sudo bash xtmc_deploy.sh
```

## 管理命令

```bash
# 启动系统
xtmc start

# 停止系统
xtmc stop

# 重启系统
xtmc restart

# 查看状态
xtmc status

# 查看日志
xtmc logs

# 备份数据
xtmc backup
```

## 配置说明

编辑 `/opt/xtmc-trading/config.env`:

```env
# 交易所配置
EXCHANGE=binance
API_KEY=your_api_key
API_SECRET=your_api_secret

# 交易配置
SYMBOL=BTCUSDT
TIMEFRAME=1h
ENABLE_TRADING=false  # 测试时保持false
MAX_POSITION=100
RISK_PERCENT=2
```

## 安全建议

1. **测试阶段**: 使用只读API密钥
2. **IP白名单**: 限制API密钥访问IP
3. **小额测试**: 不要投入超过承受能力的资金
4. **定期备份**: 使用 `xtmc backup` 定期备份
